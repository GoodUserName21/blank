# Python Environment Checker

A simple Python script to verify that Python is installed and working correctly on your system.

## What It Does

This script checks if Python is properly installed and displays the Python version you're running.

## Getting Started

### Prerequisites
- Python 3.x installed on your system

### How to Run

```bash
python check_python.py
```

or 

```bash
python3 check_python.py
```

## Expected Output

You should see a "Test" message printed to the console along with your Python version information.

## Purpose

This is a quick sanity check to ensure Python is working correctly in your environment before running more complex projects.
